## Form Background
Controls the background color and image for forms.



- Variable Name: form_background
- Data Type: background
- Default Value: 
	- units: 
	- color: #FFFFFF
	- image: 
	- repeat: 
	- x: 0
	- y: 0


[/segment]

- Data Type: border
