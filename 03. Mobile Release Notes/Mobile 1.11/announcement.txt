We are proud to announce the release of the vBulletin Mobile Suite 1.11. This version of the app fixes a wide number of customer facing issues. The Mobile Suite is available for all versions of vBulletin that support the Mobile Api.

[b]YouTube Uploading[/b]
Due to a number of changes to the API deployed by YouTube, uploading videos to the service from the Mobile Apps has been removed. We are sorry for any inconvenience this may cause.

[b]Facebook Connect[/b]
Due to changes in the Facebook API, in order to use Facebook Connect with the Mobile Apps, you must run vBulletin under SSL on your webserver. If you have any questions regarding this, please contact our support team.
