We're happy to announce the release of vBulletin 5.1.10.  All together over 151 issues have been resolved in vBulletin 5.1.10.  While most of these issues are bug fixes, there are a few enhancements to the software.

Theme Improvement
General improvements have been applied to all themes distributed with vBulletin 5 and usable on vBulletin Cloud. These include resolving display issues, improving background display and including all styles in the theme selection tool within Site Builder.

Responsive
Continued work has been undertaken in upgrading the Responsive Framework so that it is more reliable.
